Welcome to the SIMCA BACKEND

This is a simple API that allows you to manage your students and their classes.

EXAMPLES

  1. GET /student/ todos los estudiantes
  2. GET /student/:id todos los estudiantes con id = id
  3. GET /class/:idStudent todos las clases de un estudiante con id = idStudent
  4. GET /program/:idStudent todos los programas de un estudiante con id = idStudent
  5. GET /dasdadsasdasadasdasdasdasdt todos los semestres de un estudiante con id = idStudent
